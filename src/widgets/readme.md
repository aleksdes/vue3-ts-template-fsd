# Слой widgets
**widgets** (виджеты) — композиционный слой для соединения сущностей и фич в самостоятельные блоки.

Например: IssuesList, UserProfile.

Можно использовать только **в pages и app**.

Пример: **widgets** содержит "собранную" карточку поста, с содержимым и интерактивными кнопками, в которые вшиты запросы к бэкенду.
